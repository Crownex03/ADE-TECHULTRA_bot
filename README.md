# ⚡👑 ADE-TECHULTRA BOT 👑⚡

A powerful Telegram group management bot with anti-abuse, games, and admin features.

---

## 🚀 Features

### 🎮 Games
- `/truth` - Random truth question
- `/dare` - Random dare challenge
- `/riddle` - Solve a tricky riddle
- `/answer` - Reveal riddle answer

### 👑 Admin Commands
- `/kick` - Kick a member
- `/promote` - Promote to admin
- `/demote` - Demote an admin
- `/ban` - Ban a member
- `/unban` - Unban a member

### ⚠️ Warning System
- `/warn` - Warn a member
- `/resetwarn` - Reset warnings
- `/setwarn` - Set warning limit
- `/warnings` - Check your warnings

### 🔇 Mute System
- `/mute` - Mute a member
- `/unmute` - Unmute a member

### ⚙️ Group Settings
- `/welcome` - Set welcome message
- `/goodbye` - Set goodbye message
- `/onlyadmins` - Enable admin-only mode
- `/leave` - Bot leaves the group

### 📊 Info Commands
- `/groupinfo` - Group information
- `/grouplink` - Get group invite link
- `/tagall` - Tag all members
- `/setdesc` - Set group description
- `/disp` - Display group stats
- `/stats` - Show bot settings

### 🛡️ Anti-Abuse
- `/antilink` - Block links
- `/antiimage` - Block images
- `/antivideo` - Block videos
- `/antisticker` - Block stickers
- `/antiaudio` - Block audio
- `/antimention` - Block mentions
- `/antigrouplink` - Block group links

### 📩 Utility
- `/feedback` - Send feedback to owner
- `/broadcast` - Broadcast message to admins
- `/setrules` - Set group rules
- `/rules` - View group rules

### 🤖 Auto-Reply
- `/autoreply on/off` - Toggle auto-reply
- `/setreply` - Custom auto-reply message

### 🔒 Owner Commands
- `/myid` - Show owner ID
- `/restart` - Restart the bot
- `/clearcache` - Clear cache and restart
- `/calc` - Calculator
- `/update` - Auto-update from GitHub
